Der Chrome-Browser

Chrome ist ein schneller, einfacher und sicherer Browser, der für das moderne Web entwickelt wurde.

Chrome herunterladen

Geschwindigkeit

Chrome baut in jeder Hinsicht auf Geschwindigkeit. Der Browser startet im Handumdrehen, lädt Webseiten in Sekundenschnelle und führt komplexe Web-Apps blitzschnell aus.

Schneller Start

Wenn Sie auf das Chrome-Symbol auf Ihrem Desktop klicken, wird Chrome in Sekundenschnelle gestartet.

Schnelles Ausführen von Web-Apps

Chrome ist mit V8 ausgerüstet, einer noch leistungsfähigeren JavaScript-Engine, die wir entwickelt haben, um komplexe Webanwendungen blitzschnell auszuführen.

Mit jeder neuen Version steigert Chrome die Geschwindigkeit von JavaScript im Browser in Bezug auf die V8- und Kraken-Benchmarks.

Schnelles Suchen und Navigieren

In die kombinierte Such- und Adressleiste von Chrome, die sogenannte Omnibox, können Sie sowohl Suchanfragen als auch Webadressen eingeben.

Die Omnibox bietet beim Eintippen Vorschläge für Suchanfragen und Webadressen sowie eine Funktion zur automatischen Vervollständigung, damit Sie schon nach wenigen Tastenanschlägen eine Antwort auf Ihre Suchanfrage erhalten.

Schnelles Laden von Webseiten

Chrome basiert auf der Open Source-Rendering-Engine WebKit und lädt Webseiten in Sekundenschnelle.

Interessiert an den technischen Details, die Chrome so schnell machen?

Im Chrome-Comic erfahren Sie noch mehr über WebKit, die V8 JavaScript-Engine und Browsergeschwindigkeiten.

In den technischen Videos über die DNS-Vorauflösung, V8 JavaScript-Engine und DOM-Verbindungen wird erläutert, warum Chrome so schnell ist.

Einfachheit

Das Browserfenster von Chrome ist übersichtlich und leicht verständlich. Sie können beispielsweise innerhalb desselben Feldes suchen, navigieren und Tabs beliebig anordnen – ganz einfach und schnell.

Nichts verkomplizieren

Eines der Grundkonzepte von Chrome ist die Einfachheit. Es ist ein Browser, der dem Web nicht im Wege steht. Dadurch können Sie sich auf das konzentrieren, was für Sie am wichtigsten ist.

Einfache Suche und Navigation

Im Web sollte man ganz einfach ans Ziel gelangen können. Sie sollten sich nicht fragen müssen: "Suche ich im Web oder navigiere ich zu einer Website?" Sie sollten einfach anfangen, etwas einzugeben, und schon am Ziel ankommen.

That's why Chrome just has one box – the Omnibox – where you can both search the web and navigate to sites. It autocompletes as you type and offers suggestions.

Effiziente Tab-Verwaltung

Die praktischen Tabs von Chrome können leicht per Drag & Drop verschoben und neu angeordnet werden und dank der Multi-Prozess-Architektur von Chrome können Sie hundert Tabs öffnen, ohne dass Chrome dadurch langsamer wird.

Sie können außerdem Ihre Lieblings-Tabs, z. B. Ihre E-Mails, an die Tableiste anheften, damit sie jedes Mal, wenn Sie Chrome starten, an derselben Stelle angezeigt werden.

Tabulatortaste für Suche

Warum erst zu YouTube.com gehen und dort nach einem Video suchen? Geben Sie einfach youtube.com in die Omnibox ein und drücken Sie die Tabulatortaste, um direkt auf YouTube zu suchen. Sie können auch benutzerdefinierte Suchkürzel für Ihre Lieblings-Websites einrichten.

Integrierter PDF Viewer

Chrome verfügt über einen integrierten PDF Viewer, sodass Sie PDFs im Handumdrehen laden können, ohne Software oder ein Plug-in installieren zu müssen. So können Sie einfach per Mausklick PDF-Dateien anpassen, speichern und drucken.

Beginnen, wo Sie aufgehört haben

Wenn Sie Chrome schließen, werden die Tabs gespeichert, die Sie geöffnet hatten, sodass Sie später einfach dort weitermachen können, wo Sie aufgehört haben.

Sicherheit

Chrome bietet integrierten Malware- und Phishing-Schutz sowie viele weitere Sicherheitsfunktionen. Automatische Aktualisierungen sorgen dafür, dass Sie immer die neuesten Sicherheitsupdates erhalten.

Safe Browsing

Chrome zeigt Ihnen eine Warnmeldung an, bevor Sie eine Website besuchen, die im Verdacht steht, Malware- oder Phishing-Content zu enthalten.

Wenn in Chrome die Safe Browsing-Technologie aktiviert ist und Sie beim Surfen auf eine potenzielle Phishing- oder Malware-Website geraten, wird eine ähnliche Warnseite wie oben angezeigt.

Phishing findet gewöhnlich über eine gefälschte Website statt und bedeutet, dass jemand eine andere Identität vortäuscht, um Sie dazu zu bringen, persönliche oder andere vertrauliche Daten preiszugeben. Malware ist dagegen Software, die oft ohne Ihr Wissen auf Ihrem Computer installiert wird und die Ihren Computer schädigen soll oder dazu dient, Daten von Ihrem Computer zu stehlen.

Die Sandbox-Technologie

Sandbox-Technologien verhindern, dass sich Malware automatisch auf einem Computer installiert oder Aktivitäten in einem Browser-Tab dazu verwendet, die Aktivitäten in einem anderen Tab zu beeinflussen. Die Sandbox-Technologie sorgt in Ihrem Browser für zusätzlichen Schutz vor schädlichen Webseiten, die versuchen, Programme auf Ihrem Computer zu installieren, Ihre Webaktivitäten zu überwachen oder Daten von Ihrer Festplatte zu stehlen (siehe Video unten).

Automatische Updates

Damit Sie stets durch die neuesten Sicherheitsupdates geschützt sind und Chrome immer auf dem neuesten Stand ist, überprüft Chrome regelmäßig, ob Updates vorliegen. Durch die Suche nach Updates wird sichergestellt, dass Ihre Chrome-Version ohne Ihr Zutun automatisch mit den neuesten Sicherheitsfunktionen und Fehlerbehebungen aktualisiert wird.

Datenschutz

In Chrome können Sie selbst entscheiden, welche Informationen privat bleiben sollen. Gleichzeitig hilft Chrome Ihnen, die Daten zu schützen, die Sie online teilen.

Inkognito-Modus

Wenn Sie nicht möchten, dass von Ihnen besuchte Websites im Browserverlauf bzw. Ihre Downloads im Downloadverlauf aufgezeichnet werden, haben Sie die Möglichkeit, im Inkognito-Modus zu surfen. Außerdem werden alle im Inkognito-Modus erstellten Cookies gelöscht, sobald Sie alle offenen Inkognito-Fenster schließen.

Wenn Sie im Inkognito-Modus surfen, wird das Symbol für den Inkognito-Modus in der Ecke angezeigt. Der Inkognito-Modus ist sehr praktisch, da man so einfach anonym im Web surfen kann, ohne die Datenschutzeinstellungen zwischen den einzelnen Browsersitzungen ändern zu müssen. So können Sie beispielsweise gleichzeitig eine normale Sitzung und eine anonyme Sitzung in verschiedenen Fenstern durchführen.

Datenschutzeinstellungen

You can control all your privacy preferences from the Chrome menu by selecting Settings, clicking show advanced settings and then adjusting the Privacy section.

Browserdaten löschen

Mit Chrome können Sie Ihre Browserdaten jederzeit löschen, damit diese Informationen nicht in Ihrem Browser gespeichert werden, einschließlich Browser- und Downloadverlauf, Cookies, gespeicherter Passwörter und gespeicherter Formulardaten. Weitere Informationen

Datenschutzeinstellungen nach Website anpassen

In den Inhaltseinstellungen von Chrome können Sie die Datenschutzeinstellungen für Cookies, Bilder, JavaScript und Plug-ins auf Website-Ebene festlegen. Beispielsweise können Sie die Einstellungen für Cookies so festlegen, dass Chrome Cookies nur für bestimmte vertrauenswürdige Websites automatisch zulässt, und dann für alle anderen Websites Cookies manuell zulassen oder blockieren. Zusätzliche Datenschutzressourcen

Sie können das Whitepaper zum Datenschutz bei Google Chrome herunterladen, sich mit den Google Chrome-Datenschutzbestimmungen vertraut machen oder in der Chrome-Hilfe mehr über die Verwaltung Ihrer Datenschutzeinstellungen erfahren.

Personalisierung

Chrome bietet Ihnen viele Möglichkeiten, den Browser individuell anzupassen. Sie können Ihre Einstellungen optimieren und Apps, Erweiterungen und Designs aus dem Chrome Web Store hinzufügen.

Apps

Apps aus dem Chrome Web Store nutzen leistungsfähige neue Webtechnologien, um Ihnen bestmögliche Produktivität, Lernerfolge und Spielerlebnisse zu bieten, stets nur ein Klick entfernt auf der "Neuer Tab"-Seite.

Neue Nutzer hinzufügen

Und wenn Sie Ihren Computer mit anderen Leuten gemeinsam nutzen? Sie möchten wahrscheinlich nicht, dass Ihre Lesezeichen, Apps und Erweiterungen mit denen einer anderen Person vermischt werden, und auch nicht, dass Ihre Chrome-Inhalte auf allen ihren Geräten synchronisiert werden.

Jetzt können Sie neue Nutzer zu Chrome hinzufügen. So hat jeder Nutzer seine eigene Chrome-Version und kann sich in Chrome anmelden, um seine Inhalte zu synchronisieren.

Gehen Sie einfach zu "Optionen" ("Einstellungen" auf einem Mac) und klicken Sie auf "Privates" und dann auf "Neuen Nutzer hinzufügen".

Erweiterungen

Erweiterungen sind benutzerdefinierte Tools, mit deren Hilfe Sie mit Chrome noch mehr tun können, z. B. Ihre Musik verwalten, Screenshots erstellen und Websites mit Freunden teilen, unabhängig davon, wo Sie sich im Web befinden.

Designs

Es ist ganz einfach, Chrome noch ein wenig lebendiger zu gestalten. Sie können über den Chrome Web Store Designs von herausragenden Künstlern installieren oder auf mychrometheme.com Ihr eigenes Design erstellen und es mit Freunden teilen.

Anmeldung

Durch die Anmeldung in Chrome stehen Ihnen Ihre Lesezeichen, Ihr Suchverlauf und weitere Einstellungen auf allen Ihren Computern zur Verfügung. Sie sind damit auch automatisch in Ihren Google-Diensten angemeldet.

Nehmen Sie Ihr Web mit

Durch die Anmeldung in Chrome werden Ihre Lesezeichen, Apps, Ihr Verlauf und weitere Einstellungen auf alle Ihre Geräte übertragen (Video ansehen). Alle Änderungen, die Sie auf einem Gerät vornehmen, werden sofort auch auf den anderen Geräten übernommen und Ihre Chrome-Inhalte sind absolut sicher, selbst wenn Ihr Computer verloren geht oder ausfällt. Es ist Ihr Web. Nehmen Sie es mit. Rufen Sie einfach das Schraubenschlüssel-Menü auf und wählen Sie "In Chrome anmelden…" aus.
