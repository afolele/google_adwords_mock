Weshalb es wichtig ist

Ein Passwort kann leichter gestohlen werden, als man denkt

Ein Passwort kann leichter gestohlen werden, als man denkt

Bei jeder dieser häufig durchgeführten Aktionen kann Ihr Passwort gestohlen werden:

  • Sie verwenden dasselbe Passwort auf mehr als einer Website.
  • Sie laden Software aus dem Internet herunter.
  • Sie klicken auf Links in E-Mails.

Mit der Bestätigung in zwei Schritten halten Sie Angreifer fern, sogar wenn diese Ihr Passwort schon gestohlen haben.

Angenommen, Sie verlieren den Zugriff auf Ihr Konto samt aller Inhalte

Wenn Angreifer Ihr Passwort stehlen, können sie Ihnen den Zugriff auf Ihr Konto verweigern und anschließend z. B. Folgendes tun:

  • Alle Ihre E-Mails, Kontakte, Fotos usw. ansehen oder sogar löschen
  • Sich als Sie ausgeben und unerwünschte oder schädliche E-Mails an Ihre Kontakte senden
  • Ihr Konto zum Zurücksetzen der Passwörter Ihrer anderen Konten wie Bankkonten oder Shopping-Konten nutzen

Einfaches Anmelden

Während der Anmeldung können Sie angeben, dass Sie die Bestätigung in zwei Schritten auf dem jeweiligen Computer nicht mehr verwenden möchten. Anschließend brauchen Sie auf diesem Computer bei der Anmeldung nur noch Ihr Passwort einzugeben.

Der Schutz ist dennoch gegeben, da die Bestätigung in zwei Schritten weiterhin aktiviert ist, wenn Sie oder andere Nutzer versuchen, sich auf einem anderen Computer in Ihrem Konto anzumelden.

Zusätzliche Sicherheit

Die meisten Nutzer verwenden nur eine Ebene, nämlich ihr Passwort, um ihr Konto zu schützen. Mit der Bestätigung in zwei Schritten sind Sie noch besser geschützt: Wenn ein Hacker Ihr Passwort stiehlt, bräuchte er außerdem Ihr Telefon oder Ihren Sicherheitsschlüssel, um auf Ihr Konto zugreifen zu können.

Anmeldung mit Passwort und Telefon

Mit der Bestätigung in zwei Schritten schützen Sie Ihr Konto zweifach, mit Ihrem Passwort und über Ihr Telefon bzw. den Sicherheitsschlüssel.

Persönliche Bestätigungscodes

Codes sind eindeutig und werden nur für Ihr Konto erstellt, wenn Sie sie benötigen. Wenn Sie sich für die Option mit Bestätigungscodes entscheiden, werden diese per SMS an Ihr Telefon gesendet oder Sie erhalten sie per Sprachanruf oder über unsere mobile App. Jeder Code kann nur einmal verwendet werden.

Unter Funktionen erfahren Sie mehr über Sicherungsoptionen für die Fälle, wenn Sie nicht auf Ihr Telefon zugreifen können.
